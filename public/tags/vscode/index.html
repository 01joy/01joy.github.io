UbuntuдёӢдҪҝз”ЁVSCodeиҝһжҺҘGithub

VSCodeжҳҜеҫ®иҪҜејҖжәҗзҡ„дёҖдёӘеҫҲејәеӨ§зҡ„IDEпјҢеҸҜд»Ҙж”ҜжҢҒеҮ д№ҺжүҖжңүзј–зЁӢиҜӯиЁҖпјҢиҖҢдё”жҳҜи·Ёе№іеҸ°зҡ„пјҢLinuxз”ЁжҲ·з»ҲдәҺеҸҜд»Ҙз”ЁдёҠе®Үе®ҷжңҖејәIDEдәҶгҖӮжҲ‘жңҖиҝ‘еңЁдҪҝз”ЁVSCodeзј–еҶҷи°ғиҜ•PythonйЎ№зӣ®пјҢе…¶и°ғиҜ•еҠҹиғҪеҫҲејәеӨ§пјҢе’ҢVSдёҠи°ғиҜ•C++зҡ„ж„ҹи§үжҳҜдёҖж ·зҡ„пјҢејәзғҲжҺЁиҚҗгҖӮ VSCodeиҝҳеҸҜд»ҘиҝһжҺҘGithubпјҢиҝӣиЎҢзүҲжң¬жҺ§еҲ¶гҖӮдёӢйқўд»ҘжҲ‘жңҖиҝ‘еӯҰд№ зҡ„ж·ұеәҰеӯҰд№ йЎ№зӣ®дёәдҫӢпјҢд»Ӣз»ҚдёӢжҖҺж ·еңЁUbuntuдёӢдҪҝз”ЁVSCodeиҝһжҺҘGithubгҖӮд»ҘжҲ‘forkзҡ„repoдёәдҫӢпјҡhttps://github.com/01joy/neural-networks-and-deep-learningгҖӮ иҝһжҺҘGithubжңүдёӨз§Қж–№ејҸпјҢдёҖз§ҚжҳҜHTTPSпјҢеҸҰдёҖз§ҚжҳҜSSHпјҢеңЁжҜҸдёӘrepoйЎөйқўзҡ„еҸіиҫ№пјҢжңүдёҖдёӘClone or downloadжҢүй’®пјҢеҸҜд»ҘиҺ·еҸ–еҲ°иҝҷдёӨз§ҚиҝһжҺҘж–№ејҸзҡ„ең°еқҖгҖӮHTTPSж–№ејҸе’ҢзҪ‘еқҖзұ»дјјпјҢд»ҘHTTPSејҖеӨҙпјӣSSHж–№ејҸд»Ҙgit@githu.comејҖеӨҙгҖӮдҪҝз”ЁHTTPSиҝһжҺҘжҜ”иҫғз®ҖеҚ•пјҢдҪҶжҳҜжҜҸж¬Ўpushзҡ„ж—¶еҖҷйңҖиҰҒиҫ“е…Ҙз”ЁжҲ·еҗҚе’ҢеҜҶз ҒпјҢжҜ”иҫғйә»зғҰпјҢеҰӮжһңжғіи®°дҪҸеҜҶз ҒпјҢйңҖиҰҒжҠҠз”ЁжҲ·еҗҚе’ҢеҜҶз Ғд»ҘжҳҺж–Үзҡ„еҪўејҸдҝқеӯҳеҲ°дёҖдёӘж–Үд»¶дёӯпјҢдёӘдәәж„ҹи§үдёҚж–№дҫҝдё”дёҚе®үе…ЁгҖӮдёӢйқўд»ҘSSHиҝһжҺҘдёәдҫӢиҝӣиЎҢд»Ӣз»ҚгҖӮ з”ҹжҲҗдёҖеҜ№ж–°зҡ„SSHе…¬й’Ҙе’Ңз§Ғй’ҘпјҢе№¶ж·»еҠ еҲ°ssh-agentдёӯгҖӮжіЁж„Ҹз”ҹжҲҗзҡ„ж—¶еҖҷйңҖиҰҒиҫ“е…ҘpassphraseпјҢиҝҷдёӘpassphraseдёҚжҳҜGithubзҡ„еҜҶз ҒпјҢиҮӘе·ұйҡҸдҫҝеҸ–дёҖдёӘи®°дҪҸе°ұеҘҪгҖӮhttps://help.github.com/articles/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent/ жҠҠSSHзҡ„е…¬й’Ҙж·»еҠ еҲ°GithubиҙҰеҸ·дёӯпјҡhttps://help.github.com/articles/adding-a-new-ssh-key-to-your-github-account/ жөӢиҜ•SSHиҝһжҺҘжҳҜеҗҰжҲҗеҠҹпјҡhttps://help.github.com/articles/testing-your-ssh-connection/ еңЁиҜҘж–Үд»¶еӨ№еҶ…еҲқе§ӢеҢ– git remote add origin git@github.com:01joy/notes-on-writing.git # дҪҝз”Ёrepoзҡ„SSHең°еқҖ git pull origin master # жҠҠиҝңзЁӢд»Јз ҒжӢүеҲ°жң¬ең° дҝ®ж”№д»Јз Ғ git add . #
